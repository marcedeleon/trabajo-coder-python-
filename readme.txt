SISTEMA DE GENERACION DE BLOG 
CON ESTE PROGRAMA PODERMOS IR ARMANDO BLOG SEPARADOS POR USUARIOS 
PARA INSTALARLO:
1 clona el repositorio: git clone  https://github.com/marcedeleon/trabajo-coder-python-.git
2 ve al directorio del proyecto: cd sistema
3 instala las dependencias: pip install -r requierements.txt 
Uso 
El programa esta diseñado con un entorno de usuario. el cual te pedirá que estes logueado para continuar. los invitados solo podrán entrar a la pagina principal y ver las opciones de loqueo. 
Luego una vez logueado entraras al panel de usuario donde podés crear blogs, modificar blogs
Y eliminar blogs.
En cuando el usuario crea el blog queda en la pagina principal para que lo siga modificando cuando quiere.
Luego también tendrá una opción en la que puede pasar esos blogs a terminado donde se almacenan en una página diferente.
